TalentoTech - Back-End

Backend con Node.js, Express, Corse, Firebase y Swagger UI

Características principales

  • Registro y login con autenticación JWT.
  • Documentación interactiva con Swagger UI.
  • Gestión de productos (rutas protegidas).
  • Conexión a base de datos NoMySQL.
  • Arquitectura modular.

Endpoints principales

GET /api/products
POST /api/products
GET /api/products/:id
PUT /api/products/:id
DELETE /api/products/:id

Configuración rápida

  1. Cloná el repo https://github.com/amarrero86mr/talentotech_TP_backEnd.git
  2. Ejecutá npm install.
  3. Crea y configura tu Firestore/Firebase ejemplo_db.json
  4. Configurá tus variables en .env.
  5. Iniciá el servidor con npm run dev.
  6. Abrí http://localhost:3000/register y registrate
  7. Ve a logueate http://localhost:3000/login
  8. Compia en la consola el token localStorage.token
  9. Y pega el token en docs para acceder sin limite a los endpoint
